About

CommonCounsel AI is a concept site demonstrating how to present an AI-assisted legal information product responsibly — with clear disclaimers and guardrails.

Mission

Make legal language more understandable, especially for people dealing with stressful situations. Focus on clarity and information — not advice.

  • Explain terminology and process in plain English
  • Help users prepare better questions for a licensed attorney
  • Encourage safety and privacy by default

Design principles

  • Boundaries first: not a lawyer, not advice
  • Structure: summaries, checklists, questions
  • Accessibility: keyboard shortcuts, readable layout
  • Performance: static pages optimized for GitHub Pages
