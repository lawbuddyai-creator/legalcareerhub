# CommonCounsel AI (GitHub Pages)

A Virginia-focused, guardrails-first legal information and document understanding site built with Next.js + TypeScript + Tailwind, exported statically for GitHub Pages.

## Local dev
```bash
npm install
npm run dev
```

## Deploy
Push to `main`. GitHub Actions builds and deploys to GitHub Pages.

## Notes
- Dark mode default with toggle (saved to localStorage).
- Demo pages are front-end only and do not provide legal advice.
